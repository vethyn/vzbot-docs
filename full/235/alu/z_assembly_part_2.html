4.3 Top rod mounts

Material Quantity Note
M4 10mm 4 -
M4 20mm 4 -

Important: When mounting the top rod's make sure the top rod with the extra hole for the z switch is in the back

Mount the 10mm M4 screws and t-nuts to all top rod mounts and loosely turn in the 20mm M4 screws in the clamping mechanism.
Mount all top rod mounts to the rod first, finally tighten the rod holders to the frame with the pre-inserted M4 hardware.



After installing check the movement of the bed assembly by turning the leadscrews. This should operate smoothly. If not, place the bed assembly in the top most position, untighten all top rod holders and retighten to align the rods.

4.4 Single z motor

Material Quantity Note
M3 10mm 6 -
M4 10mm 2 -
GT2 20 teeth 1 -
NEMA 17 stepper 1 -


Mount the bracket shown below to the frame using the 10mm M4 screws and T-nuts. Make sure it is nicely centered in the middle of the frame.



Mount the stepper motor to the top bracket using the 10mm M3 screws. Attach the motor pulley like shown below.



Mount the previous assembly to the bracket on the frame using 10mm M3 screws.


With the motor in the furthest position, loop the belt around the motor pulley and tighten the belt. Be sure to check if the belt is horizontally parallel to the frame for smoothest operation.

4.5 Z-Switch & bed chain

Material Quantity Note
M3 10mm 4 -
M3 8mm 1 For microswitch
M2 10mm 2 For microswitch
M3 40mm 1 -
M3 nut 3 -
Microswitch 1 -
Chain mount ( printed ) 1 link
Endstop mount ( printed ) 1 link


Mount the printed chain mount to the bed frame using 2x10mm M3 screws (Vz-2235) or 4x6mm M3 (Vz-330). Then mount the cable chain using 10mm M3 screws and nuts like shown below.


Mount the microswitch to the printed adapter using M2 screws. Then mount the assembly to the top rod holder using the 8mm M3. Finally screw in the 50mm M3 screw and nut like shown below. After setting the correct Z-height using the adjustment screw use the M3 nut to lock the screw in place in order to make sure the screw doesn't drift off position.
